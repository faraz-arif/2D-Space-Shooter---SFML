#### **SFML Game Project – Semester 1**

This is a 2D game built using C++ and SFML as part of my first semester programming fundamentals project at FAST-NU, Lahore.

#### **Features**

1. Player movement.

2. Collision detection.

3. Score system and game saving.

4. Simple UI.

#### Technologies Used

1. C++.

2. SFML Library (3.0.2).

#### Note About Assets

All images and sounds used in this project were taken from free online resources and are used only for educational purposes.

#### How to Run

1\. Install SFML on your pc. 

2\. Link SFML libraries.  

3\. Compile and Run.
